## Fixing Common Confluence Setup Issues

- Trouble finding a confluence page? Check the space location, page permissions, search filters, and whether the page sits under another parent in the confluence wiki.  
- Jira confluence links not appearing? Confirm that jira and confluence are connected through the same Atlassian organization and that users have access to both projects.  
- Confluence cloud access problems? Review account permissions, browser sessions, single sign-on settings, and workspace invitation status.  
- Confluence templates missing? Ask a space admin to verify template availability, global template permissions, and whether the right space is selected.  
- Confluence api or confluence rest api requests failing? Confirm authentication, endpoint format, user permissions, rate limits, and the target confluence documentation path.
